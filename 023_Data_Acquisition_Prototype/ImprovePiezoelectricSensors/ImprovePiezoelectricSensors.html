Phoenix Ambulatory Blood Pressure Monitor Project
Sub-project: Improve Piezoelectric Sensors

Project Description

The purpose of the Improve Piezoelectric Sensors subproject is to answer questions about the suitability of using the piezoelectric sensors and amplifier circuits built in previous projects, then improve them, providing a valuable input to the Data Acquisition Device Prototype project.

Those questions include:

  • Can we get the accuracy needed to meet the ANSI SP10 standard for blood pressure monitors?
  • How close together can the sensors be?
  • Can we use the inside of the upper arm so it's close to the heart?  Is that even necessary?
  • Can we get to where the wearer doesn't have to stop breathing?  (See Akhila's paper)
  • What materials and mechanism do we need to provide better skin-to-sensor contact to improve the signal-to-noise ratio?
  • What changes do we need to the amplifier circuit to prevent clipping of the signal in the presence of high blood pressures?
The project would start with the piezoelectric sensors and amplifiers built in previous projects and, using equipment in the UMN IEEE student center lab, attempt to answer the questions above.
