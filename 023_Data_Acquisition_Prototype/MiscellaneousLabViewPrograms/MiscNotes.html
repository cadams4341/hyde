Phoenix Ambulatory Blood Pressure Monitor Project
Sub-project: Data Acquisition Prototype

Project Description

This directory contains some LabView Virtual Instruments that have been used to collect data in the lab from early 2010 (from Jomar).

We are  using the National Instruments USB-6009 Data Acquisition (DAQ) system.
We were using SignalExpress V3.0.
We are currently using Labview V8 with NI-DAQmx9.0.2 Labview drivers for Win2007/XP.

NI only supports certain versions (mostly old versions) of certain Linux distros. More details are at http://digital.ni.com/public.nsf/websearch/35D2B5E483A001A18625716B005BE464?OpenDocument

This might be an alternative to Labview via an open source alternative.
Measurement Computing.  DAQFlex Open-Source Software Framework.
http://www.mccdaq.com/solutions/DAQFlex-Solutions.aspx
