Phoenix Ambulatory Blood Pressure Monitor Project
Sub-project: Power Sources

 

Project Description

The purpose of the power source project is to find suitable power devices for the ambulatory blood pressure monitor.


Project Deliverables

  • Lists of batteries with data
  • Lists of potential alternative power sources (solar, etc.).
