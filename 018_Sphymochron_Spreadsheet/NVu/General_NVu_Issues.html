Some general issues with NVu

Problems:
It does not handle HTML file names (or any othe r file name) with spaces in the name correctly.  This is mostly an FTP issue, actually.  However, NVu not only allows to create files with names that won't work right, it actually encourages it by giving your new HTML files a name derived from the page title that you enter when you save the file for the first time.  If you type in a title with spaces in it that's pleasant for humans to read, the resulting default file name will cause problems later.
